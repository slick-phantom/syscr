# Lesson 3. Operations in crystal

## What you will learn 
- Maths with numbers
- Combining Texts
- Comparing Values
- Logical Operations

## Math operation
crystal support basic maths just like a calculator 

### Addition(+)
```crystal
points = 10 + 7 # 17
total = 100 + 37  #137
```
it can also be used with 
```crystal
points = 10
points += 7   #17
```
### subtraction
```crystal
remaining = 20 - 10   #10
balance = 100 - remaining #90
```

### multiplication 
```crystal
area = 5 * 8   #40
total = area * 2   #80
```

### division 
```crystal
half = 10/2   #5.0(note returns float)
share = 100/3  #33.33333
```

### Modulo (%) - remainder
```crystal
remainder = 10 % 3  #1
is_even = 10/2 #0
```
### Exponent
```crystal
square = 20 ** 2
cube = 3 ** 3
```
